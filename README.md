# Digital Panel Inspection Application

A local, offline digital inspection tool for electrical/control panels.

## Features (Current Stable Version)
- Upload BOM (PDF)
- Auto-generate inspection checklist
- Accurate PART DESCRIPTION extraction
- Select checklist item to view full BOM details
- Works fully offline (local FastAPI + frontend)

## Tech Stack
- Backend: Python, FastAPI, pdfplumber
- Frontend: HTML, CSS, JavaScript
- Platform: Local (WSL / Desktop)
